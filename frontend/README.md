## Build Setup 

``` 
 # clone
git clone https://github.com/leif-sh/fog.git
```

```
# cd
cd  fog/frontend
```

```
# install dependencies
npm install
```

```
# Compiles and hot-reloads for development
npm run dev
```

```
# Compiles and minifies for production
npm run build
```
